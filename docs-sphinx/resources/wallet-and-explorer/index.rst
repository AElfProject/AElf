Wallet and Block Explorer
=========================

Explorer
--------

`Github <https://github.com/AElfProject/aelf-block-explorer>`__

Currently, the explorer provides functions such as viewing blocks,
transactions, purchasing resources, voting and node campaigning as well
as viewing contracts.

iOS/Android Wallet
------------------

iOS/Android Wallet provides basic asset management and cross-chain
trading capabilities. It also provides an open application platform for
developers to access the wallet according to the usage document of the
wallet SDK.

Web Wallet
----------

`Github <https://github.com/AElfProject/aelf-web-wallet>`__

The Web Wallet provides basic transaction related functionality.
