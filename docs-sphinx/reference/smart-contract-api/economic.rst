AElf.Contracts.Economic
-----------------------

Economic contract.

The Economic contract establishes the economic system of the AElf. When
the block chain starts to work, this contract will initialize other
contracts related to economic activities.

Implement AElf Standards ACS1.

This transaction only can be send once because after the first sending, its state will be set to initialized.   |

Note that this will override all fees of the method.                                                                                                                         |
